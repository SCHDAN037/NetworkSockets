This is your project home

File images to be sent on terminal must be located on same directory as makefile

Received images are located in the file_database directory

Chat Terminal commands:
----------------------
quit chat:
  Enter   :q  OR  :quit

send image:
  Enter   :send /path-to-file/filename

reply to image offer:
  Enter  :Y  to accept OR  :N  to reject
